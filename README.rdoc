
* Ruby version
 - ruby 2.2.3p173

* System dependencies
	You should install the follow:
 - Rails 4.2.4 (http://rubyonrails.org/download/)
 - Postgres 9.4 (http://postgresapp.com)

* Configuration
 1. Git clone project
 2. Run $ bundle install (Install Gems Dependencies)
 3. Run $ rake db:schema:load (Database Creation)

* Run Project
 4. Run $ Rails s
